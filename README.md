# Automating Webform using Selenium Junit.
## About this project:
## Automated site link: https://www.digitalunite.com/practice-webform-learners
## Here I automate the following features:
- Handle Alerts
- Write on textbox
- Handle CheckBoxRadio
- Scroll
- Select Date
- Upload Image
- Handle Checkbox
- Handle Submit Button
- Assertions Handling
